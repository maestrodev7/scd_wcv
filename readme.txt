== DOCUMENTATION AND USAGE ==

Visit our documentation section on our website : [SCD DOCUMENTATION] (https://www.gajelabs.com/scd-documentation/)

= For marketplaces or Multi vendors sites =
 1- Go to "WooCommerce -> SCD Currencies -> CURRENCIES SETTINGS"
 2- Check "Multi roles based, webpages display currency" option then save changes.
 3- Each vendors/SHOP MANAGER can select his products prices in each currency he can handles.

= Add "SCD - Smart Currency Detector" Widget (SCD Widget) =

 1- Go to the admin view under "Appearance > Widgets" and select the SCD Widget 
 2- Choose a area in your website where you want to put in the widget. 
 3- Go back to your website: you can see the widget at the related place and your customers can start setting their own currency.
